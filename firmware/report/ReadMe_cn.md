# ReadMe\_cn

### 1. 介绍 <a href="#1.-introduction" id="1.-introduction"></a>

&#x20;    我们提供TCP数据上报功能，实时获取手臂的位置，速度等信息；端口30000，频率为200HZ；下面简单介绍如何使用数据上报获取并解析数据。
